###Discord Bot connected to CoinmarketCap & Cryptowat.ch API

Simple Chat Bot for Price Updates, written in nodejs.

```markdown
## Installation

1) npm install

2) Launch the bot: $ node bot.js
    
3) Commands: .help, .ticker, .ticker {bfx,gdax,polo}   
   ```
